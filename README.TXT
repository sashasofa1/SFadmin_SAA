Все сервисы развернуты с помощью Ansible за исключением OpenVPN-server и OpenVPN-client.
Стек ELK развернут с помощью rpm пакетов хранимых в директории /etc/ansible/packages/
Для серверов на ubuntu было использовано зеркало yandex репозитория elastic.
Все роли самописные, в процессе изучения приобретен опыт использования loop.
В связи с этим качество ролей разнится.
